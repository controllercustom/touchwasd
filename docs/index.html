Bootloader Mode Instructions


            

                
M5Stack AtomS3

                
    
                    
  1. Connect the AtomS3 to your computer via USB-C
    2. 
                    
  2. Press and hold the side button (the only button on the side)
    3. 
                    
  3. Keep holding for 2 seconds until the green LED inside turns on
    4. 
                    
  4. Release the button
    5. 
                    
  5. The device is now in download mode (green LED indicates bootloader mode)
    6. 
                    
  6. Click Flash Firmware above and select the device in the browser serial prompt
    7. 
                

            


            

                
Generic ESP32-S3 Dev Module

                
    
                    
  1. Connect the board to your computer via USB
    2. 
                    
  2. Press and hold the BOOT button (GPIO0)
    3. 
                    
  3. While holding BOOT, press and release the EN / RST button
    4. 
                    
  4. Release the BOOT button
    5. 
                    
  5. The device is now in download mode
    6. 
                    
  6. Click Flash Firmware above and select the device in the browser serial prompt
    7.
